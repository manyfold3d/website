This release focuses on federation of some social features. We've improved how new model posts appear on other services like Mastodon, and you can now quote those messages as well! We also count up likes from other servers for the original model, letting you know just how popular your work is!

## What's Changed
### ✨ New Features ✨
* Include incoming federated likes in displayed like count by [@Floppy](https://github.com/Floppy) in [#5922](https://github.com/manyfold3d/manyfold/pull/5922)
* Allow quoting for federated create/update notes by [@Floppy](https://github.com/Floppy) in [#5937](https://github.com/manyfold3d/manyfold/pull/5937)
* Respond to Quote Authorizations and automatically allow for new/update messages by [@Floppy](https://github.com/Floppy) in [#5967](https://github.com/manyfold3d/manyfold/pull/5967)
### 🔒 Security 🔒
* Force multiuser mode if federation is enabled by [@Floppy](https://github.com/Floppy) in [#5939](https://github.com/manyfold3d/manyfold/pull/5939)
### 🐛 Bug Fixes 🐛

* Fix accidentally-removed translation keys by [@Floppy](https://github.com/Floppy) in [#5929](https://github.com/manyfold3d/manyfold/pull/5929)
* Removed unused translation by [@Floppy](https://github.com/Floppy) in [#5931](https://github.com/manyfold3d/manyfold/pull/5931)
* More translation fixes by [@Floppy](https://github.com/Floppy) in [#5940](https://github.com/manyfold3d/manyfold/pull/5940)
* Fix structure of canQuote interactionPolicy by [@Floppy](https://github.com/Floppy) in [#5961](https://github.com/manyfold3d/manyfold/pull/5961)
* Fix compatibility notes by [@Floppy](https://github.com/Floppy) in [#5962](https://github.com/manyfold3d/manyfold/pull/5962)
