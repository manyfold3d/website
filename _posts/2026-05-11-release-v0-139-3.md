Another bundle of bugfixes, both around collection relationships and a number of other areas.

## What's Changed
### 🐛 Bug Fixes 🐛
* Ensure model<->collection relations are unique by [@Floppy](https://github.com/Floppy) in [#6086](https://github.com/manyfold3d/manyfold/pull/6086)
* Strip whitespace from fediverse handles by [@Floppy](https://github.com/Floppy) in [#6091](https://github.com/manyfold3d/manyfold/pull/6091)
* Fix collections disappearing on model name or preview change by [@Floppy](https://github.com/Floppy) in [#6092](https://github.com/manyfold3d/manyfold/pull/6092)
* Prevent 3d rendering of image files by [@Floppy](https://github.com/Floppy) in [#6093](https://github.com/manyfold3d/manyfold/pull/6093)
* Ignore file extensions when working out file groupings by [@Floppy](https://github.com/Floppy) in [#6096](https://github.com/manyfold3d/manyfold/pull/6096)
* Fix page error when "uploading" with no actual files by [@Floppy](https://github.com/Floppy) in [#6106](https://github.com/manyfold3d/manyfold/pull/6106)
* Save datapackages when collection/link associations change by [@Floppy](https://github.com/Floppy) in [#6103](https://github.com/manyfold3d/manyfold/pull/6103)
